## NCCL Communication Summary

| Metric | Value |
|--------|-------|
| AllGather avg (per layer) | 22.0 ms |
| ReduceScatter avg (per layer) | 22.9 ms |
| NCCL % of wall time | 3.2% |
| Typical GEMM kernel duration | 45–162 ms |
| Compute-to-communication ratio | ~30:1 |

`ModuleWrapPolicy({LlamaStyleDecoderLayer})` shards each decoder layer
independently, allowing PyTorch FSDP to overlap the AllGather for layer N+1
with the backward compute of layer N.
The result is that NCCL collectives account for only **3.2% of wall time**
against 45–162 ms GEMM kernels — a compute-to-communication ratio of ~30:1.
